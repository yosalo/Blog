# 为什么使用 Jekyll

* 模板不用自己找  
* 写东西比较检索，简化时间  

# 如何安装 Jekyll
* 安装Ruby(这里一定要勾选添加到环境变量PATH)
* 安装RubyGems
* 安装gem
* 安装jekyll-paginate
* 如果提示任何错误,安装提示的包即可

# 如何使用jekyll
* jekyll new blog #创建你的站点
* jekyll serve 	  #启动你的http服务 
* jekyll build    #编译站点
* jekyll serve build --detach #后台启动服务
